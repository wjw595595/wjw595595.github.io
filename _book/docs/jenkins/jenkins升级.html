jenkins升级

jenkins的war包更新比较快，升级的话有两种方式


1.自动升级

在jenkins右下角可以看到当前版本号


现在版本是2.176.2


在管理中有一个新版本提示，可以直接点击自动升级，如下图所示


然后他会进入到自动更新中，勾选安装后重启，等待安装好后就可以看到版本变更了


安装好后重新登录可以看到在管理中有个恢复成上一个版本，升级成功



2.手动升级

下载最新版本的war包，将包放到服务器war包所在位置，替换原本的war包，然后重新启动就好

如果找不到服务器war包


找到war包

1.在管理中查看系统信息


可以在系统信息里，找到相应的位置


2.执行Linux系统命令 find

find / -name jenkins.war



下载新版本包


点击下载，然后传到服务器上，或者可以直接在服务器上下载

[root@localhost ~]# wget http://updates.jenkins-ci.org/download/war/2.176.3/jenkins.war



停止jenkins服务



备份原本的war包

进入到文件夹内

[root@localhost ~]# cd /usr/local/jenkins/

[root@localhost jenkins]# mv jenkins.war jenkins.war.bak

[root@localhost jenkins]# mv /root/jenkins.war .


启动jenkins

[root@localhost jenkins]# java -jar jenkins.war --httpPort=9091 &
