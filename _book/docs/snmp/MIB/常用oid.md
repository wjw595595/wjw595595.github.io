## cisco

```
1.3.6.1.4.1.9.2.1.56.0  #cpu 5秒使用率
```

### Cisco NX-OS/ACI

```
[[inputs.snmp.field]]
    name = "CPU1m"
    oid = "1.3.6.1.4.1.9.9.109.1.1.1.1.7.1"
[[inputs.snmp.field]]
    name = "MemUsed"
    oid = "1.3.6.1.4.1.9.9.109.1.1.1.1.12.1"
[[inputs.snmp.field]]
    name = "MemFree"
    oid = "1.3.6.1.4.1.9.9.109.1.1.1.1.13.1"
```
