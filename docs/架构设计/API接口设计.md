# 接口设计

json格式

```json
{
  #返回状态码
  code:integer,    
  #返回信息描述
  message:string,
  #返回值
  data:object
}
```

code状态码

| 分类 | 区间                           |
| ---- | ------------------------------ |
| 1**  | *#1000～1999 区间表示参数错误* |
| 2**  | *#2000～2999 区间表示用户错误* |
| 3**  | *#3000～3999 区间表示接口异常* |
