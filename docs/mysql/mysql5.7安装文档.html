方法一：yum安装  

1. 下载mysql的repo源 （ https://dev.mysql.com/downloads/repo/yum/）

###5.7

$ wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm  

###8.0

$ wget http://repo.mysql.com/mysql80-community-release-el7-3.noarch.rpm


###8.0

rpm -ivh mysql80-community-release-el7-3.noarch.rpm

###5.7

rpm -ivh mysql-community-release-el7-5.noarch.rpm  


###执行结果：会在/etc/yum.repos.d/目录下生成两个repo文件：mysql-community.repo 和 mysql-community-source.repo


###更新yum

yum clean all

yum makecache


##查看yum版本

yum repolist all | grep mysql


根据具体需要：

禁用：yum-config-manager --disable mysql80-community

启用：yum-config-manager --enable mysql57-community


yum-config-manager 这个命令在：  yum -y install yum-utils


或者：修改

cat /etc/yum.repos.d/mysql-community.repo  enable字段


安装：mysql

yum install mysql-community-server


然后启动

开启mysql 服务

systemctl start mysqld.service

获取初始 密码

cat /var/log/mysqld.log | grep password


登录： mysql -uroot -p


登录后修改密码：

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';


授权

create user 'root'@'%' identified with mysql_native_password by '123456';

grant all privileges on *.* to 'root'@'%' with grant option;

flush privileges;


mysql> set global validate_password.policy=0;  ###密码长度问题

方法二：离线安装

下载 .rpm-bundle rpm包


第一步：下载tar包 https://dev.mysql.com/get/Downloads/MySQL-8.0/mysql-8.0.15-1.el7.x86_64.rpm-bundle.tar


第二步：yum install -y lrzsz


第三步：

在/usr/local目录下新建目录mysql

cd /usr/local

mkdir mysql


第四步：

在/usr/local/mysql目录下使用rz -be命令上传mysql-8.0.15-1.el7.x86_64.rpm-bundle.tar


第五步：解压安装 4个rpm包

tar -xvf mysql-8.0.15-1.el7.x86_64.rpm-bundle.tar


安装：mysql

rpm -ivh mysql-community-common-8.0.15-1.el7.x86_64.rpm --nodeps --force 命令安装 common


依次安装

libs,client,server rpm 包 mysql-community-libs-compat-5.7.29-1.el7.x86_64（这个需要，不然报错）


查看：rpm -qa | grep mysql


第六步：

初始化

mysqld --initialize;

[root@1234 mysql]# mysqld --initialize;

mysqld: error while loading shared libraries: libaio.so.1: cannot open shared object file: No such file or directory

yum install -y libaio安装依赖


mysqld --initialize;

chown mysql:mysql /var/lib/mysql -R;

systemctl start mysqld;


cat /var/log/mysqld.log | grep password查看密码


登录后修改密码：

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '123456';


授权

create user 'root'@'%' identified with mysql_native_password by '123456';

grant all privileges on . to 'root'@'%' with grant option;


GRANT ALL PRIVILEGES ON . TO 'root'@'%' IDENTIFIED BY '123456' WITH GRANT OPTION;



flush privileges;




LogFile=/var/log/zabbix/zabbix_server.log


systemctl start zabbix-server.service


Zabbix-server报错 找不到libmysqlclient.so.18 解决：mysql-community-libs-compat-5.7.29
