## Cloud Toolkit

Cloud Toolkit提供轻量级devops的idea plugin（Alibaba Cloud Toolkit），可以让你快速部署远程服务期（支持ssh、docker）、可以配置跳板机，让你在idea在本地一键部署、同时可以通过shell来管理远程主机。
