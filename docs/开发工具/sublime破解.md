# sublime破解

## 步骤

下载sublimeText3的安装包并安装（已经安装的可以忽略）

- 安装包安装
- host文件：127.0.0.1   license.sublimehq.com  （hosts文件地址:C:\Windows\System32\drivers\etc）
- 破解文件复制到sublime安装目录
- 管理员运行破解文件，复制注册码
- 打开sublime -> help -> enterlicense 输入注册码
- 恭喜你完成破解
