修改保存路径



 npm config set cache "D:\nodejs\node_cache"  
 npm config set prefix "D:\nodejs"  --nodejs会自动寻找该路径下的node_modules文件夹为实际存放全局模块的路径，这也是为啥叫prefix不叫global的原因；以后安装的全局模块都会被放到D:\Program Files\nodejs\node_modules下，跟npm模块在一个文件夹中




npm重新安装node_modules方法

 

1. 安装rimraf ： cnpm install rimraf -g

2. 执行： rimraf node_modules 删除文件荚 

3. 清空缓存： npm cache clean --force

4. 重新安装淘宝镜像： npm install -g cnpm --registry=https://registry.npm.taobao.org

5. 再次执行：cnpm i

 

 

npm其经的操 作
