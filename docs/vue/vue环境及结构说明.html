一、配置环境：

1、安装nodejs

https://nodejs.org/en/ 下载稳定版


2、查看版本： node -v；node自带 npm npm -v


3、安装淘宝镜像

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v 


4、安装 webpack ：npm install webpack -g  ; webpack -v


5、安装vue-cli 脚手架

npm install vue-cli -g ; vue -V


-----------------------------------------


二、构建vue项目

新建一个文件夹 vuetest

cd e:/vue_test

vue init webpack vuetest
