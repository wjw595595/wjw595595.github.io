Oracle自带的用户

Oracle安装完毕创建数据库实例的时候，会自动生成三个用户sys，system，scott。

sys用户是超级管理员，具有最高权限，充当sysdba角色，可以执行create database，默认密码是mannager。

system是操作管理员，仅次于sys，没有create database权限，其他权限与sys相同，默认密码是change_on_install.

scott用户为普通用户，默认密码是tiger，可以用system去解锁。

一般数据库维护使用，使用system即可。



解锁；

sqlplus / as sysdba


alter user system account unlock;
